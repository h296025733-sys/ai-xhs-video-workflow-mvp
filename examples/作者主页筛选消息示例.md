# 作者主页筛选消息示例

这些示例可以放进 OpenClaw、钉钉、飞书或本地 `message.txt` 里测试。

## 最近 7 天 + 点赞门槛

```text
找小红书号 YOUR_XHS_ID，最近7天内点赞100以上的视频，取5条。
BGM 用韩国 TikTok OOTD 热门歌，从第20秒开始。
先 dry-run 看看筛选结果。
```

## 作者主页链接直传

```text
处理这个作者主页最近30天点赞200以上的视频，取8条。
BGM 用日本 TikTok 穿搭热门歌，从第15秒开始。
https://www.xiaohongshu.com/user/profile/YOUR_CREATOR_ID
```

## 只下载原视频，不换 BGM

```text
找小红书号 YOUR_XHS_ID，最近15天内点赞50以上的视频，取10条。
只下载原视频，不换BGM，保留原声。
```

## 多 BGM 版本方向

```text
找小红书号 YOUR_XHS_ID，最近7天内点赞100以上的视频，取3条。
每条视频生成3个不同 BGM 版本：
1. 韩国 TikTok OOTD 热门歌
2. 日本穿搭 lookbook 热门歌
3. 日韩 fashion outfit viral song
从第20秒开始。
```

多 BGM 版本在公开版里保留了脚本骨架，真实效果依赖 BGM 搜索结果和本机下载环境，建议生成后人工试听复核。
