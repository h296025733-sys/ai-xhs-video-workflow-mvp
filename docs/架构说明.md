# 架构说明

## 总体思路

项目把“自然语言任务”和“视频处理脚本”分开：

- OpenClaw / 企业 IM 负责接收中文需求。
- 包装脚本负责解析链接、BGM、dry-run、起始秒数、输出目录等意图。
- FastAPI 负责创建任务、管理素材、调用 FFmpeg。
- 本地 outputs 目录保存结果，不进入 Git。

## 核心模块

| 模块 | 作用 |
| --- | --- |
| `app/main.py` | FastAPI 应用入口 |
| `app/quick_bgm/router.py` | quick-bgm API 和本地演示页面 |
| `app/quick_bgm/media.py` | FFmpeg / ffprobe 调用 |
| `app/quick_bgm/store.py` | 本地 JSON 状态与 BGM 库 |
| `app/quick_bgm/xhs_adapter.py` | 单条小红书链接导入适配 |
| `app/quick_bgm/xhs_creator_importer.py` | 作者主页和作品预览适配 |
| `scripts/quick_bgm_automation.py` | 通用 CLI |
| `scripts/openclaw_quick_bgm.py` | 自然语言消息包装脚本 |
| `scripts/openclaw_xhs_author_direct_v2.py` | 作者主页批量处理脚本 |

## 关键边界

公开版不包含真实登录态、真实素材、真实 OpenClaw 配置。作者主页最近 N 条依赖登录态或外部采集工具，本仓库只保留接入位置和错误兜底。
