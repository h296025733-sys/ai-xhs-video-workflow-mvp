# 第三方依赖与致谢

本项目是一个工作流整合项目，重点是把多个成熟工具接成一个能跑的本地流程。公开仓库没有内置第三方项目源码，也没有内置任何真实账号、cookie、token 或素材。

## XHS-Downloader

- 项目：JoeanAmier/XHS-Downloader
- 地址：https://github.com/JoeanAmier/XHS-Downloader
- 作用：作为小红书作品解析、下载、API/CLI/MCP 能力的可选外部依赖。
- 本仓库处理方式：不复制其源码，只提供适配环境变量和调用方式。

使用前请阅读 XHS-Downloader 自身的 README、许可证和平台合规说明。公开版示例里只出现 `YOUR_XHS_DOWNLOADER_PATH`、`http://127.0.0.1:5556` 这类占位配置，不包含真实 cookie。

## Playwright

- 项目：Microsoft Playwright
- 地址：https://github.com/microsoft/playwright
- 作用：通过 Chrome CDP 连接已登录浏览器，完成小红书作者搜索、页面滚动、DOM/网络响应采集。

这里的关键设计是复用本机浏览器登录态，而不是把 cookie 写入仓库。

## FFmpeg

- 项目：FFmpeg
- 地址：https://ffmpeg.org/
- 作用：视频探测、音频替换、封面抽帧、音轨混合。

FFmpeg 是本项目视频处理的底层工具。不同平台安装方式不同，建议把 `ffmpeg` 和 `ffprobe` 加到系统 PATH。

## FastAPI

- 项目：FastAPI
- 地址：https://fastapi.tiangolo.com/
- 作用：提供本地 Web 页面、quick-bgm API、批量任务接口。

## yt-dlp

- 项目：yt-dlp
- 地址：https://github.com/yt-dlp/yt-dlp
- 作用：BGM 搜索下载链路的可选工具。

注意：BGM 版权和平台规则需要使用者自行确认。本项目只提供技术链路，不鼓励绕过版权或平台规则。

## OpenCV / PySceneDetect

- OpenCV：https://opencv.org/
- PySceneDetect：https://www.scenedetect.com/
- 作用：视频画面分析、场景切分、关键帧/封面辅助处理。

这些能力在当前 MVP 中属于“素材理解与复核辅助”，不是深度算法训练项目。

## 本项目许可证说明

本仓库只对本项目自有代码选择许可证；第三方工具按照各自项目的许可证使用。本项目没有把 XHS-Downloader、FFmpeg、Playwright 等第三方源码复制进仓库。
