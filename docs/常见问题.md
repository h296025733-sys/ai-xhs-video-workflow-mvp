# 常见问题

## 这是完整自动发布系统吗？

不是。当前公开版是本地 MVP 和工作流骨架，不包含 TikTok 自动发布。

## 为什么作者主页不能保证自动抓取？

小红书作者主页最近 N 条通常需要登录态、Cookie、MCP、浏览器脚本或外部下载器支持。公开版不会内置这些私人登录信息。

## 没有 XHS-Downloader 能用吗？

可以。你仍然可以上传本地视频和本地 BGM，使用 FFmpeg 完成换 BGM。

## BGM 搜索失败怎么办？

可以手动上传 BGM，或换更明确的搜索词，例如“韩国 TikTok OOTD 热门歌 2026”。

## 为什么不提交 outputs？

outputs 里通常是处理后视频、封面、报告和日志，可能包含真实素材、个人信息或版权内容，不适合进入公开仓库。
