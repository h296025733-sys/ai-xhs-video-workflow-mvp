# 快速开始

建议按“三步走”验证：先跑本地视频换 BGM，再跑单条小红书链接，最后再跑作者主页自动化。

## 1. 安装依赖

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m playwright install chromium
```

只验证本地视频换 BGM 时，Playwright 可以先不装；要跑小红书作者主页自动化时再装。

## 2. 检查 FFmpeg

```powershell
ffmpeg -version
ffprobe -version
```

如果命令不可用，请先安装 FFmpeg，并把 `ffmpeg.exe`、`ffprobe.exe` 加到 PATH。

## 3. 准备配置

```powershell
Copy-Item .env.example .env
```

`.env` 只放你自己的本机配置，不要提交。只跑本地视频时，可以先保持：

```text
XHS_DOWNLOAD_MODE=disabled
XHS_CREATOR_IMPORT_MODE=manual_links
```

## 4. 启动本地服务

```powershell
.\scripts\run_api.ps1
```

浏览器打开：

```text
http://127.0.0.1:8004/quick-bgm/page
```

上传一个本地视频和一个 BGM，点击生成。如果这一步成功，说明基础链路已经通了。

## 5. 命令行处理本地视频

```powershell
.\.venv\Scripts\python.exe .\scripts\quick_bgm_automation.py `
  --api-base http://127.0.0.1:8004 `
  --video-path "D:\workspace\demo\input.mp4" `
  --bgm-strategy local_random `
  --run
```

## 6. 单条小红书作品链接

先启动你自己的 XHS-Downloader API，然后配置：

```powershell
$env:XHS_DOWNLOADER_API_BASE_URL = "http://127.0.0.1:5556"
$env:XHS_DOWNLOAD_MODE = "api"
```

再运行：

```powershell
.\.venv\Scripts\python.exe .\scripts\quick_bgm_automation.py `
  --api-base http://127.0.0.1:8004 `
  --note-url "https://www.xiaohongshu.com/explore/YOUR_NOTE_ID" `
  --bgm-strategy search_download `
  --bgm-query "韩国 TikTok OOTD 热门歌 2026" `
  --start 20 `
  --run
```

## 7. 作者主页自动化

先启动带调试端口的 Chrome：

```powershell
& "C:\Program Files\Google\Chrome\Application\chrome.exe" `
  --remote-debugging-port=9222 `
  --user-data-dir="D:\workspace\xhs-cdp-profile"
```

在这个 Chrome 里登录小红书，然后执行：

```powershell
$env:XHS_CDP_URL = "http://127.0.0.1:9222"

.\.venv\Scripts\python.exe .\scripts\openclaw_xhs_author.py `
  --api-base http://127.0.0.1:8004 `
  --message "找小红书号 YOUR_XHS_ID，最近7天内点赞100以上的视频，取5条，BGM用韩国 TikTok OOTD 热门歌，从第20秒开始，正式生成"
```

想先看解析结果，不打开浏览器：

```powershell
.\.venv\Scripts\python.exe .\scripts\openclaw_xhs_author.py `
  --message "找小红书号 YOUR_XHS_ID，最近7天内点赞100以上的视频，取5条" `
  --parse-only
```
