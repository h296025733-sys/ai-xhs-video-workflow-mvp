# 快速开始

本项目优先面向 Windows 本地工作流，Linux/macOS 也可按 Python + FFmpeg 的方式改造。

## 1. 安装依赖

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## 2. 检查 FFmpeg

```powershell
ffmpeg -version
ffprobe -version
```

如果没有安装，可先安装 FFmpeg，并把 `ffmpeg.exe` 和 `ffprobe.exe` 加入 PATH。

## 3. 准备配置

```powershell
Copy-Item .env.example .env
```

只做本地上传视频换 BGM 时，可以先保持 `XHS_DOWNLOAD_MODE=disabled`。

## 4. 启动服务

```powershell
.\scripts\run_api.ps1
```

浏览器打开：

```text
http://127.0.0.1:8004/quick-bgm/page
```

## 5. 命令行处理

```powershell
.\.venv\Scripts\python.exe .\scripts\quick_bgm_automation.py `
  --api-base http://127.0.0.1:8004 `
  --video-path "D:\workspace\your-project\demo.mp4" `
  --bgm-strategy local_random `
  --run
```

小红书链接处理需要你自行配置 XHS-Downloader 或其它合法采集方式。
