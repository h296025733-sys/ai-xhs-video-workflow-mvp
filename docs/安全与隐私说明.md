# 安全与隐私说明

## 不应提交的内容

- `.env` 和任何真实配置。
- API key、token、cookie、Authorization、Bearer。
- OpenClaw workspace、模型配置、渠道配置。
- 小红书登录态、浏览器 profile、session。
- 真实视频、音频、BGM、封面、成品视频。
- 运行日志、报错日志、下载缓存。
- 公司、客户、同事、老板等真实业务信息。

## 推荐做法

- 所有密钥只放在本机环境变量或私有配置文件中。
- 示例配置只使用 `YOUR_..._HERE` 占位符。
- 输出目录放在 `outputs/` 或其它被 `.gitignore` 排除的位置。
- 上传 GitHub 前运行敏感信息扫描和大文件扫描。
- 如果历史文件里出现过真实 token，建议去对应平台手动作废或重置。

## 版权与平台规则

本项目只提供本地处理工作流示例。实际使用小红书素材、BGM 或 TikTok 发布时，需要遵守平台规则、版权授权和公司内部合规要求。
