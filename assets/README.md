# assets 公开素材目录

这个目录只放可以公开展示的轻量示例素材，例如脱敏后的界面截图、流程示意图或占位说明。

真实运行时产生的视频、音频、封面和下载缓存都不应该放在这里。它们属于本地工作产物，默认应保存在 `outputs/`、`.runtime/` 或你自己的素材目录中，并由 `.gitignore` 排除。

不要放入：

- 真实小红书视频。
- 成品视频。
- BGM 音频。
- 客户或公司素材。
- 带个人信息的截图。
- 浏览器缓存、登录态或下载器运行文件。

如果后续需要展示界面效果，请使用脱敏后的本地演示截图，并确认画面里没有账号、头像、昵称、手机号、公司路径、真实订单或真实业务数据。
